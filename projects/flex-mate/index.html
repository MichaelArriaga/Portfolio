Problem

Amazon flex, a gig delivery app created by Amazon Logistics Inc. Does not have any way to show users an accurate fuel cost and travel estimate to each of the warehouses which is a significant amount of common un-accounted for expenses that delivery partners incur while making their deliveries.

Solution

Create an app that uses Distance Matrix data to estimate fuel cost for travel to each delivery station based on the user provided average cost per gallon (cpg) as well as their vehicle average miles per gallon (mpg).

Features

Flex Mate App

  • Fuel Cost Estimation
  • Show Real-Time Travel/Traffic Estimations
  • Show Job Types By Station
  • Coverage for as many Flex Regions as possible
